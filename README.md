# 📧 Email Sender

A simple and efficient platform to **streamline your communication** by sending emails to multiple recipients at once.
Built with a clean interface, this app makes it easy to compose, manage, and send emails effortlessly.

![EmailSenderDemo](https://github.com/KrishBharadwaj5678/EmailSender/raw/main/EmailSenderDemo.png)

## ✨ Features

| Feature               | Description                                    |
| --------------------- | ---------------------------------------------- |
| 📩 Bulk Email Sending | Send emails to multiple recipients at once     |
| ✍️ Easy Composition   | Simple interface to write and format emails    |
| ⚡ Fast & Efficient    | Quickly deliver emails with minimal setup      |
| 🎯 User-Friendly UI   | Clean and intuitive design using Streamlit     |
| 🔐 Secure Sending     | Uses SMTP protocol for reliable email delivery |

---

## 🛠️ Tech Stack

| Technology       | Purpose                                  |
| ---------------- | ---------------------------------------- |
| 🐍 **Python**    | Core programming language                |
| 🎈 **Streamlit** | Builds the interactive web interface     |
| 📧 **smtplib**   | Handles sending emails via SMTP protocol |

## 🚀 Getting Started

### 1️⃣ Clone the Repository

```bash
git clone https://github.com/KrishBharadwaj5678/Attendly.git
```

### 2️⃣ Navigate to the Project

```bash
cd EmailSender
```

### 3️⃣ Install Dependencies

```bash
pip install -r requirements.txt
```

### 4️⃣ Run the App

```bash
streamlit run app.py
```
